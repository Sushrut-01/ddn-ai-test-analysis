WHY NO DATA?
============
The app is running but needs backend API to load data.

SOLUTION IN 3 STEPS
===================

STEP 1: Start Backend API
--------

Open Command Prompt / PowerShell and run:

   cd C:\DDN-AI-Project-Documentation\implementation
   pip install flask flask-cors python-dateutil
   python dashboard_api_full.py

   ✓ Backend starts on http://localhost:5006
   ✓ API serving test failure data
   ✓ Keep this running while testing app


STEP 2: Update App Settings (if needed)
----------

1. Open the app on your Android device
2. Go to SETTINGS (bottom tab)
3. Look for "API Endpoint"
4. If it shows "192.168.1.7" (old IP), change it:

   How to find YOUR computer IP:
   - Windows: Open CMD, type: ipconfig
   - Look for "IPv4 Address" (e.g., 192.168.1.5)
   
   Update endpoint to: http://[YOUR_IP]:5006
   (Example: http://192.168.1.5:5006)

5. Save settings


STEP 3: Restart App
--------

1. Close and reopen the app
2. Wait 2-3 seconds for data to load
3. YOU SHOULD NOW SEE:
   ✓ Dashboard with list of test failures
   ✓ Analysis tab with AI insights
   ✓ Analytics with charts
   ✓ And more!

TROUBLESHOOTING
===============

Still No Data?
   1. Check backend is running (see terminal output)
   2. Verify API endpoint in app Settings
   3. Confirm device on same WiFi as computer
   4. Close and reopen app
   5. Wait 3-5 seconds for data to load

API Endpoint Error?
   1. Check computer IP: ipconfig (Windows) or ifconfig (Mac/Linux)
   2. Update endpoint in Settings to: http://[YOUR_IP]:5006
   3. Restart app

Backend Won't Start?
   1. Make sure Python 3 installed
   2. Install Flask: pip install flask flask-cors
   3. Navigate to correct folder
   4. Run: python dashboard_api_full.py

Can't Connect to Backend?
   1. Device on same WiFi as computer?
   2. Computer IP correct in Settings?
   3. Firewall blocking port 5006?
   4. Check logs: adb logcat | grep flutter
