🔧 TROUBLESHOOTING
==================

Problem: "App not installed"
Solution:
   - Check Android version (5.0+)
   - Delete old version: adb uninstall com.example.ddn_ai_mobile
   - Reinstall: adb install -r ddn_ai_mobile_release.apk

Problem: App crashes on open
Solution:
   - Check internet connection
   - Verify API endpoint in Settings
   - Try: adb shell pm clear com.example.ddn_ai_mobile
   - Reinstall app

Problem: No data showing
Solution:
   - Check API endpoint in Settings
   - Enable internet connection
   - Restart the app
   - Data will cache for offline use

Problem: No notifications
Solution:
   - Check Settings > Notifications enabled
   - Device needs Android 5.0+
   - Ensure Google Play Services installed
