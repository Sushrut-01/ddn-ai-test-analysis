╔═══════════════════════════════════════════════════════════════════════════╗
║                  HOW TO START POSTGRESQL - SIMPLE GUIDE                   ║
╚═══════════════════════════════════════════════════════════════════════════╝

YOU GOT THIS ERROR: "System error 5 has occurred. Access is denied."
WHY: Windows needs Administrator privileges to start services


╔═══════════════════════════════════════════════════════════════════════════╗
║                         SOLUTION - OPTION 1                               ║
║                  (EASIEST - Use the batch file)                           ║
╚═══════════════════════════════════════════════════════════════════════════╝

Step 1: Find the file
   └─> START-POSTGRESQL.bat

Step 2: RIGHT-CLICK on the file
   └─> DO NOT double-click!
   └─> Right-click with your mouse

Step 3: Select "Run as administrator"
   └─> Click "Yes" when Windows asks permission

Step 4: You should see:
   [OK] PostgreSQL service started successfully!


╔═══════════════════════════════════════════════════════════════════════════╗
║                         SOLUTION - OPTION 2                               ║
║                  (Use Windows Services Manager)                           ║
╚═══════════════════════════════════════════════════════════════════════════╝

Step 1: Press [Windows Key] + [R] together
   └─> The "Run" dialog will open

Step 2: Type: services.msc
   └─> Press Enter

Step 3: Find "postgresql-x64-18" in the list
   └─> Scroll down to find it

Step 4: Right-click on "postgresql-x64-18"
   └─> Click "Start"

Step 5: Status should change to "Running"


╔═══════════════════════════════════════════════════════════════════════════╗
║                         SOLUTION - OPTION 3                               ║
║                    (Use Command Prompt)                                   ║
╚═══════════════════════════════════════════════════════════════════════════╝

Step 1: Press [Windows Key]

Step 2: Type: cmd

Step 3: RIGHT-CLICK on "Command Prompt"
   └─> Select "Run as administrator"
   └─> Click "Yes" when asked

Step 4: Type this command and press Enter:
   net start postgresql-x64-18

Step 5: You should see:
   "The postgresql-x64-18 service was started successfully."


╔═══════════════════════════════════════════════════════════════════════════╗
║                      VERIFY IT'S WORKING                                  ║
╚═══════════════════════════════════════════════════════════════════════════╝

After starting PostgreSQL, verify it's running:

Open Command Prompt (doesn't need admin) and run:
   sc query postgresql-x64-18

Look for this line:
   STATE              : 4  RUNNING
                            ^^^^^^^
                            This should say "RUNNING"
