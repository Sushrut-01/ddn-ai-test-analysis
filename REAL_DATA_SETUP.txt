   1. Open Terminal
   2. Type: ifconfig

TROUBLESHOOTING
===============

App Still Shows No Data?

   Check 1: Is IP Address Correct?
   - Open Settings in app
   - Verify endpoint is: http://[YOUR_IP]:8005/api
   - Make sure no typos
   - Restart app
   
   Check 2: Device on Same WiFi?
   - Computer and device on SAME WiFi network?
   - Not on different networks?
   - Both connected to Internet?
   
   Check 3: Correct IP?
   - Open CMD/Terminal
   - Windows: ipconfig
   - Mac/Linux: ifconfig
   - Copy-paste IP exactly
   
   Check 4: Backend Running?
   - Backend is in Docker container
   - Should be running automatically
   - Check Docker logs if needed

Still Issues?
   1. Try different IP format (sometimes needs just IP without /api)
   2. Restart Docker containers: docker-compose restart
   3. Check firewall not blocking port 8005
   4. View logs: adb logcat | grep flutter
