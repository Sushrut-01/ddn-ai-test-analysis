QUICK INSTALLATION
==================

METHOD 1: Using ADB (Recommended)
---------------------------------
1. Enable USB Debugging on your Android device:
   - Settings > About Phone > Tap "Build Number" 7 times
   - Settings > Developer Options > Enable "USB Debugging"

2. Connect device via USB and run:
   adb install C:\DDN-AI-Project-Documentation\ddn_ai_mobile_release.apk

3. Launch app on your device (DDN AI icon)

METHOD 2: Direct Installation
------------------------------
1. Transfer ddn_ai_mobile_release.apk to your Android device
2. Open file manager and tap the APK file
3. Follow installation prompts
4. Grant requested permissions
5. Tap "Open" or find app in app drawer

METHOD 3: Android Studio Emulator
----------------------------------
1. Start Android Emulator from Android Studio
2. Run: adb install ddn_ai_mobile_release.apk
3. App appears in emulator home screen

TROUBLESHOOTING
===============

APK Won't Install
- Make sure device is Android 5.0 or higher
- Try: adb install -r ddn_ai_mobile_release.apk
- Check device has 100MB free storage

App Crashes on Open
- Check internet connection
- Go to Settings and verify API endpoint
- Restart app
- Clear app data: adb shell pm clear com.example.ddn_ai_mobile

No Data Showing
- App needs API connection first time
- Check Settings for API endpoint configuration
- Ensure internet connection enabled
- Data will cache for offline use

Notifications Not Working
- Check Settings > Notifications enabled
- Device needs Google Play Services
- Restart app
- Check Android version (needs Android 5+)
